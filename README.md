## Project Description

TaskGrid is a frontend-only web application that helps teams organize, track, and visualize tasks efficiently. Built using modern web technologies, it provides a clean and intuitive interface for collaboration and project management without requiring any backend setup.

### Core Functionality

- Dashboard overview with real-time stats
- Team management using modals
- Kanban-style task board with drag and drop
- Search and filtering by team or priority
- Progress analytics and charts
- Contact form with validation

### Technical Features

- Responsive and mobile-first design
- Modern dark UI with gradients and animations
- Client-side validation
- JavaScript-driven interactivity
- SEO-friendly and accessible structure

## Installation & Run Steps

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A local server for full functionality

### Quick Start

1. Download or clone the project
2. Navigate to the project directory
3. Open `index.html` in your browser

### Development Setup

```bash

python -m http.server 8000


npx serve .


php -S localhost:8000


```

### No-Server Method

You can open `index.html` directly, but some features may not work fully without a local server.
